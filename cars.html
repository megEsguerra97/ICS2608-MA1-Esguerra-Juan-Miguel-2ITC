Honda Type R Cars

About Cars

NSX Type R

Manufactured a very small quantity of NSX Type R cars for Japan between 1992 and 1995.
Major modifications include a more aggressive suspension and a significant weight reduction
from the standard NSX weight of 1,350 kg to 1,230 kg (2,712 lb) (2,976 lb). To save weight,
the NSX Type R was designed for the track and was devoid of sound deadening, audio, electric
windows, and air conditioning.

Integra Type R

A 1.8-liter DOHC VTEC in-line 4-cylinder (B18C) engine powers the Integra Type R. This engine
in the JDM Type R generates 200 PS (197 horsepower; 147 kW) @ 8,000 rpm.

The focus of the Type R was to minimize weight while enhancing rigidity. The Type R has larger
strut tower bars, rear performance rods on the rear frame and numerous body reinforcements.
The Type R had revised springs and dampers as well as larger sway bars.

Accord Type R

Using the CH1 Accord chassis, the Honda Accord Type-R (ATR) was built from 1998 to 2002 and offered
in European markets. There were only 1980 units made of this model.

powered by a 2.2 liter, four-cylinder, normally aspirated, DOHC, H22A7 "Red Top" VTEC engine that,
in the JDM model, generates 217 bhp (220 PS; 162 kW), and in the EDM model, 209 bhp (212 PS; 156 kW)
at 7,200 rpm and 164 lb-ft (222 Nm), of torque, at 6,700 rpm. Features include a firmer suspension
and chassis, a Helical limited-slip differential, twin-piston brakes, a dual exhaust system, 17-inch
alloy wheels, Xenon headlights, Recaro seats, and a Momo steering wheel with leather trim distinguish
the Type-R Accord model.

Civic Type R

The 1997 EK9 Civic was third to be given the Type R badge. Based on the EK4 SiR chassis, it features a
Type R prepared B16B engine producing 185 PS (182 bhp; 136 kW) along with a stiffer chassis, upgraded
sway bars and strut bars, Recaro seats trimmed in Alcantara, 15-inch alloy wheels and a large boot spoiler.
Since then, most generations have offered a Type R variant. The first generation Type R version of the Civic
was the only version not offered outside Japan.

The brand-new Honda Civic Type-R, produced at their Swindon plant, was introduced in July 2015. The vehicle's
direct-injected turbocharged 2.0 liter VTEC K20C1 gasoline engine, which is built in the United States at Honda's
Anna Engine Plant in Anna, Ohio, provides higher performance. Redesigned in 2017 but using the same engine,
the fifth generation Civic Type R can accelerate from 0 to 60 mph (97 to 97 kilometers per hour) in 4.9 seconds
and can reach a top speed of 273.6 km/h (170 mph).
